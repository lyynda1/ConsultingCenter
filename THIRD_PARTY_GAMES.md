# Third Party Games References

This project integrates/adapts ideas and logic from open-source game repositories for educational use.

## Referenced sources

1. 2048
- Repository: https://github.com/jperedadnr/Game2048
- Usage: JavaFX board/gameplay patterns adapted to local implementation.

2. Chess rules engine
- Repository: https://github.com/bhlangonijr/chesslib
- Usage: legal move generation, board state, mate/draw detection.

3. Pac-Man
- Repository: https://github.com/armin-reichert/pacman-javafx
- Usage: JavaFX Pac-Man movement/rendering concepts adapted to local implementation.

## Notes
- Only required logic/assets were adapted.
- Full external build systems were not embedded.
- Local code was adjusted for the Advisora app structure.
